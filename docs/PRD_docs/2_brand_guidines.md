# SharanAI — Brand Guidelines

## Product Name

**SharanAI**

## Tagline

**Smarter Hiring, Better Decisions**

---

# Brand Personality

SharanAI represents a modern AI-first SaaS platform built for recruiters and hiring teams.

The product should communicate:

* Intelligence
* Trust
* Simplicity
* Precision
* Speed
* Professionalism

The interface should never feel cluttered or overly decorative.

---

# Design Philosophy

SharanAI follows a minimalist product design inspired by modern SaaS platforms.

Design inspiration includes:

* Linear
* Vercel
* Arc Browser
* Notion
* Ashby
* Stripe Dashboard

The application should feel premium, lightweight, and productivity-focused.

---

* Logo

* Progress Bars

# Spacing

Use an 8px spacing system.

Examples:

* 8px
* 16px
* 24px
* 32px
* 40px
* 48px

Avoid arbitrary spacing values.

---

# Border Radius

Buttons:

12px

Cards:

20px

Inputs:

12px

Dialogs:

24px

---

# Shadows

Use subtle shadows only.

Cards should appear elevated but not floating excessively.

Avoid heavy drop shadows.

---

# Icons

Use only **Lucide React** icons.

Do not mix multiple icon libraries.

---

# Motion

Animations should be subtle and purposeful.

Allowed:

* Fade In
* Slide Up
* Scale on Hover
* Smooth Page Transitions
* Skeleton Loading

Avoid:

* Bounce animations
* Flashing effects
* Excessive motion

---

# Theme Switching

Support:

* Light
* Dark
* System

The application should automatically follow the user's operating system preference when System mode is selected.

---

# Accessibility

Ensure:

* WCAG-compliant contrast ratios
* Keyboard navigation
* Focus states
* Screen reader compatibility
* Responsive layouts

---

# Responsive Design

Desktop-first.

Support:

* Desktop
* Laptop
* Tablet
* Mobile

No functionality should be lost on smaller screens.

---

# UI Principles

* Clean layouts
* Consistent spacing
* Minimal distractions
* High readability
* Strong visual hierarchy
* Data-first presentation

---

# UI Restrictions

Do **not** use emojis anywhere in the interface.

Do **not** use clipart, cartoon illustrations, or playful graphics.

Avoid skeuomorphic elements.

Avoid excessive glassmorphism.

Do not overload the interface with colors.

Maintain a consistent and professional enterprise appearance.

---

# Brand Vision

SharanAI should feel like an enterprise-ready AI recruitment platform that organizations can trust for hiring decisions while remaining approachable and intuitive for everyday recruiters.
