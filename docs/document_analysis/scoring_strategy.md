# Candidate Scoring Strategy

Final Score =

35% Semantic Match
25% Skill Match
20% Experience Match
20% Behavioral Match

---

Semantic Match

- Candidate Summary
- Candidate Headline
- Career Descriptions

Compared Against:

- Required Skills
- Domain Expertise
- Keywords

---

Skill Match

- Skill Presence
- Skill Proficiency
- Skill Duration
- Endorsements

---

Experience Match

- Years Of Experience
- Industry Match
- Job Title Match
- Company Type

---

Behavior Match

- Recruiter Response Rate
- GitHub Activity
- Interview Completion
- Offer Acceptance
- Profile Completeness

---

Penalty Conditions

- Consulting-only Career
- Low Recruiter Response Rate
- No Production Experience
